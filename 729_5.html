Les Classes de mots

Type Catégories grammaticales
Variable nom, adjectif, déterminant, verbe, pronom
Invariable adverbe, préposition, conjonction, interjonction, onomatopée

le Nom

Catégorie grammaticale regroupant les mots qui désignent soit une espèce ou un représentant de l’espèce (noms communs), soit un individu particulier (noms propres), et qui assurent, dans la proposition, des fonctions syntaxiques spécifiques (sujet, complément d’objet, etc.).
Larousse

le Nom

  • type commun ou propre
    • chat, grammaire, café + déterminant
    Québec, Martine, Jupiter (maj) (+ déterminant)
  • caractéristiques morphologiques
    • simple / complexe - café / ras-le-bol
    féminin / masculin
    singulier / pluriel
  • caractéristiques sémantiques
    • animé / inanimé - chat / table
    humain / non-humain - étudiante / chat
    concret / abstrait - pomme / liberté
    comptable / non-comptable - pomme / eau
    individuel / collectif - étudiante / foule

l'Adjectif

Mot que l’on peut adjoindre au substantif pour exprimer une qualité de l’être, de l’objet ou de la notion désignée (adjectif qualificatif ou, simplement, adjectif), ou pour introduire ce substantif dans le discours (adjectif déterminatif).
Larousse

l'Adjectif

  • types
    • qualifiant (+ adverbe intensité) - (très) rapide
    classifiant (ø intensité) - chimique
    ordinal ou de rang - premier
    interrogatif - quelle
    participe (passé, présent) - fatigant, fatigué
    couleur - vert
  • caractéristiques morphologiques
    • simple / complexe - composé / dérivé
    masculin / féminin
    singulier / pluriel

l'Adjectif
différence épithète / attribut

  • attribut : adjectif séparé du nom par un verbe d'état
    • complément du verbe
    le ciel est bleu
  • épithète : adjectif qui fait parti du groupe nominal
    • complément du nom
    le ciel bleu

le Déterminant

Membre d’une classe de morphèmes grammaticaux qui se placent devant le nom pour l’introduire dans le discours, tout en lui apportant diverses spéci- fications. (Ce sont les articles, les adjectifs possessifs, démonstratifs, indéfinis et numéraux.)
Larousse

le Déterminant

  • référent = nom
    • genre et nombre
  • types
    • défini - le, la, les, l'
    contracté - du, des, au, aux
    indéfini - une, un, de, tel
    partitif - du, de la
    quantitatif - aucun, quelques, tout
    possessif - mon, ton, notre
    démonstratif/déictique - ce, cette, ces
    interrogatif/exclamatif - quelle
    numéral
    relatif - lequel, laquelle

le Pronom

Mot qui réfère à un mot préalablement exprimé ou à la réalité extralinguistique et dont les fonctions syntaxiques sont identiques à celles du nom.
Larousse

le Pronom

  • caractéristiques sémantiques
    • pronom de reprise - il, qui, la
    pronom nominal - je, tu
  • caractéristiques morphologiques
    • simple ou complexe - je/le leur
  • types
    • personnel - personne grammaticale
      formes conjointes - sujet : je / complément : me, te, le, la
      formes disjointes - lui, moi, toi
    prosessif - le mien, la tienne
    démonstratif - celle-ci, celui-là
    relatif - qui, que, quoi, dont
    indéfini - aucun, tous, rien, personne
    interrogatif - qui, quoi
    numéral

le Verbe

Catégorie grammaticale qui regroupe l’ensemble des formes composées d’une base lexicale et d’un certain nombre d’affixes pertinents variant en nombre, en personne, en temps, dont la fonction syntaxique est de structurer les termes de l’énoncé, et dont le rôle sémantique est de décrire les actions, les états, les modifications relatifs aux éléments auxquels réfèrent les noms sujets.
Larousse

le Verbe

  • d'action / d'état
    • agir / être
  • types
    • transitif direct - écrire, manger, lire
    transitif indirect - aller (à), douter (de)
    intransitif - éternuer, dormir
    impersonnel - pleuvoir, falloir
    pronominal - se laver, se parler, se faire, se méfier

l'Adverbe
précision du sens d'un verbe, adjectif, autre adverbe

  • de manière
    • bien, mal, vite
  • de lieu
    • ici, là, partout
  • de temps
    • hier, demain, souvent
  • de quantité
    • très, peu, assez
  • de négation
    • ne... pas, ne... jamais

la Préposition
introduit un complément dans une phrase

à, dans, par, pour, en, avec, de, chez, sous, sur

la Conjonction
relie des mots ou des propositions

  • conjonctions de coordination
    • Mais où est donc Ornicar ?
  • conjonctions de subordination
    • que, lorsque, parce que, si, bien que

l'Interjonction
pour exprimer une émotion

Oh ! Ah ! Zut ! Bravo !

l'Onomatopée
retranscrit un bruit ou un son

Cric crac, boum, miaou, plouf

Structuration(s) de la phrase

phrase = unité syntaxique

Composants

  • Groupe Sujet (GS)
    • accomplit ou subit
  • Groupe Verbal (GV)
    • verbe (+ complément)
  • Complément du verbe
      • COD (direct) - après le verbe
      COI (indirect)
      COS (second)
  • Complément Circonstanciel (CC)
    • temps, lieu, manière, but, cause

Différentes structures

  • Phrase simple
    • 1 verbe conjugué
  • Phrase complexe
      • proposition indépendante : segment autonome
      proposition principale qui commande
      proposition subordonnée qui dépend de la principale

Temps et accords

Accords
relation entre classes de mots

  • Verbe et sujet : nombre et personne
    • sujet collectif : accord selon le sens
  • Adjectif : nombre et genre
    • sauf adjectif de couleur composé = invariable
    et adjectif dérivé de nom (orange, marron
Temps Exemple
Présent Je parle
Imparfait Je parlais
Passé simple Je parlai
Futur simple Je parlerai
Passé composé J'ai parlé
Plus-que-parfait J'avais parlé
Passé antérieur J'eus parlé
Futur antérieur J'aurai parlé

Groupes de verbe
pour la conjugaison

  • 1.er groupe (-er) : réguliers
    • chanter, aimer, parler
  • 2.eme groupe (-ir + PP en -issant) : réguliers
    • finir, choisir, etc.
  • 3.eme groupe (-ir, -oir, -re, etc.) : irréguliers
    • partir, voir, prendre

Modes
1. Indicatif
faits réels ou certains

  • Présent
    • action en cours, énonciation, habitude, vérité générale ou futur proche
  • Passé composé
    • action passée achevée en lien avec le présent
    AUX au présent + PP
  • Imparfait
    • action passée inachevée, description, habitude passée, action en cours subitement interrompu
    Base présent + terminaison [-ais, -ais, -ait, -ions, -iez, -aient]
  • Plus-que-parfait
    • action antérieure à une autre action passée, hypothèse
    AUX imparfait + PP
  • Passé simple
    • action ponctuelle et achevé
    1er groupe : [-ai, -as, -a, -âmes, -âtes, -èrent]
    2eme groupe : [-is, -is, -it, -îmes, -îtes, -irent]
    3eme groupe : irréguliers

Modes
1. Indicatif
faits réels ou certains

  • Futur simple
    • action certaine future, ordre, conseil, hypothèse, supposition
    INF + terminaison [-ai, -as, -a, -ons, -ez, -ont]
  • Futur antérieur
    • action antérieure à une action future, supposition dans le passé
    AUX au futur simple + PP

Modes
2. Subjonctif
action incertaine, souhait, nécessaire

  • Présent + terminaison [-e, -es, -e, -ions, -iez, -ent]

Modes
3. Conditionnel
action soumise à condition, hypothèse, politesse

  • Futur simple + terminaison [imparfait]

Accord du participe passé
selon l'auxiliaire

  • + Avoir
    • Accord si COD avant le verbe
    Les fleurs que j’ai cueillies sont belles
    si pas COD : invariable
    J’ai cueilli des fleurs
    sauf si le pronom est COD d'un verbe infinitif
    les marmottes que j'ai vu abattre étaient mignonnes
    pas si le pronom est acteur de l'action du verbe à l'infinitif
    les marmottes que j'ai vues manger se cachent

Accord du participe passé
selon l'auxiliaire

  • + Être
    • Accord avec le sujet
    même si l'auxiliaire est sousentendu
    la nuit est tombée; les musiciens invités

Accord du participe passé
selon le verbe

  • + Verbe impersonnel
    • Toujours invariable
  • + Verbe pronominal

    • Accord
    elle s'est absentée
    sauf si le COD est placé après le verbe
    elle s'est blessé la jambe
    cas particulier invariable si pronom réfléchi est COI et ø COD
    elles se sont parlé
    sauf pour 6 verbes :
    se complaire, se déplaire, se plaire, se rire : ils se sont complu à faire ça
    s'écrier, s'exclamer (toujours accord) : elle se sont écriées

Accord du participe passé
cas particulier

  • + Verbe intransitif devenant transitif
    • Accord si le COD est placé devant
    ses plus belles années, il les a vécues ici
    sinon le verbe intransitif est invariable
    les deux années qu'il a vécu ici ont été ses plus belles

Accord du participe passé
nouvelle règle

  • Invariable quand avec avoir
  • Invariable quand avec se laisser
    • elle s'est laissé mourir

Ponctuation

Rythme et sens de la parole
intention, pause, structuration par segment

Structure de la phrase
pour les règles de ponctuation

  • thème + foyer d'information
    • ce dont on parle + ce que l'on dit de ce dont on parle

la Virgule
cas d'emplois

  • entre le sujet et le prédicat
    • Les élèves seront sanctionnés
    Les élèves qui sont arrivés en retard et qui n'ont pas apporté leur matériel, seront sanctionnés
  • avant un verbe à sujets juxtaposés
    • Le vent, la pluie, le froid, empêchent toute sortie aujourd’hui.
  • entre de courtes phrases juxtaposées
    • le ciel est bleu, la vie est belle
  • à l'intérieur d'une énumération verticale
    • + point final
    listes enchâssées : + point virgule
  • après un complément indirect en début de phrase
    • Grâce à son travail acharné, il réussit brillamment son examen
  • avec une incidente/incise, interjonction
    • Ce projet, j’en suis convaincu, sera une réussite

la Virgule
cas particulier

  • avec comme, ainsi que, autaut que, de même que, etc.
  • avec et ce
  • avant etc. et après si la phase n'est pas finie
  • avant mais
  • avant car
  • après or

la Virgule
cas particulier

  • avant voire
  • avec c'est-à-dire, autrement dit, à savoir, par exemple, entre autres
  • après les adverbes
  • après donc, puis, ensuite, sinon, autrement

la Virgule
pas de virgule

  • entre deux ni
  • pas avec non... mais, non pas... mais, non seulement... mais
    • QUAND il sépare un verbe et des compléments
    Je l’ai choisi non pas pour son prix mais pour sa qualité

le Point
cas d'emploi

  • à la fin d'une phrase
    • assertion ou interrogation
  • avant le complément du verbe ou entre groupes de même rôle
    • Il a acheté des légumes. Les tomates, les pommes de terre.
  • avant et, mais, car
  • à la fin d'une dédicade, slogan, inscription, titre

le Point
cas particulier

  • après les mots Remarques, Note, Chapitre, Annexe, Appendice
    • terme + point + tiret long
  • pour certaines abbréviations

le Point-virgule
cas d'emploi : séparer des propositions indépendantes

  • entre deux énoncés structurés identiques
  • entre des énoncés associés logiquement

les Points de suspension
cas d'emploi

  • à la fin d'une énumération comme points de suite ou d'un énoncé inachevé
  • à la place de etc.
  • pour atténuer des mots
    • jurons ou insultes
  • entre crochets pour omission dans citation
  • pour un sentiment ou une émotion

les Deux-points
cas d'emploi

  • pour annoncer un exemple, une explication
  • avant des paroles rapportées au discours indirect
  • dans un dialogue
  • avant une énumération
  • pour indiquer un lien de cause à effet
  • entre heure / minute, minute / seconde

les Guillemets
cas d'emploi

  • pour les paroles rapportées d'un discours direct
  • début et fin d'un dialogue
    • peut être à partir de la seconde réplique
  • dans citation
    • sauf citation longue
  • guillemets enchâssés

les Guillemets
cas d'emploi

  • pour les mots cités / italique
    • surnom
  • pour les mots étrangers, familiers
  • pour les mots déformés, néologismes
  • pour l'ironie
  • pour un titre

les Parenthèses
cas d'emploi

  • isoler une information non-nécessaire
  • dans adresse postale, diplôme ou grade
  • pour la traduction d'un mot
  • pour les formules d'usage
    • sic, nous soulignons, note du traducteur de la direction
  • pour encadrer une ponctuation forte
  • parenthèses enchâssées
  • pour les didascalies
  • fermante pour l'énumération

le Tiret
cas d'emploi

  • tiret long comme parenthèse
    • tiret fermant s'efface devant une poinctuation finale ou forte
  • dans parenthèses
  • pour séparer deux éléments
    • pause
  • dans dialogues
  • dans une énumération
  • après formule d'usage
    • PS, remaque, annexe, chapitre, fig.

le Point d'interrogation
cas d'emploi

  • fin de phrase interrogative
  • pour un doute ou une ellipse
  • doublé/triplé pour une émotion
  • avec point d'exclamation pour émotion vive

le Point d'exclamation
cas d'emploi

  • fin de phrase exclamative
  • après interjonction
  • doublé/triplé pour une émotion
  • avec point d'interrogation pour émotion vive

les Crochets
cas d'emploi

  • particularités dans manuscrits
  • pour redonner un contexte
  • pour les points de suspension

les Chevrons
cas d'emploi

  • pour signaler un segment vide

la Barre oblique
cas d'emploi

  • symbole de division
  • et/ou
  • opposer/rapprocher deux termes
  • pour des retours à la ligne en poésie
  • dans une url

l'Astérisque
cas d'emploi

  • renvoi à un appel de note
  • renvoi à une entrée dans l'index
  • pour atténuer un mot
    • comme les points de suspension
  • devant un H aspiré

Espaces
types

  • Espace sécable/justifiante
    • après un mot, un signe
  • Espace insécable /ou/ fine
    • après/avant une ponctuation forte
    espace fine disponible dans les logiciels d'éditique

Espaces
et ponctuation

  • Virgule : rien avant, sécable après
    • insécable après décimaux
  • Point : rien avant, sécable après
  • Point-virgule : espace fine/insécable avant, sécable après
  • Points de suspension : rien avant, sécable après
  • Deux-points : insécable avant, rien après
    • rien avant si dans heures/minutes

Espaces
et ponctuation

  • Guillemets français : insécable intérieur, sécable extérieur
  • Parenthèses : rien intérieur, sécable extérieur
  • Tiret
    • simple : sécable avant et aprèse
    long : insécable intérieur et/ou extérieur
  • Point interrogation/exclamation : insécable avant, sécable après
  • Crochets : rien intérieur, sécable extérieur
