élément essentiel de l'édition savante

Plusieurs types formels
critères

  • Temporalité
    • rétrospective : publications passées
    courante : sources nouvelles - mise à jour
  • Détail
    • signalétique : informations essentielles pour identifier la source
    sommaire : + description courte des ouvrages
    analytique : + description plus longue (neutre et objective)
    critique : + description longue avec un avis
    matériel : + caractéristiques éditoriales (bibliologie)
  • Lien aux sources
    • primaire : sources primaires
    secondaire : sur une sous-thématique
    tertiaire : bibliographie de bibliographies
  • Organisation
    • systématique : toutes les sources liées par une caractéristique éditoriale
    thématique : regroupement selon un sujet
  • Ambition
    • exhaustive : signaler tout ce qui existe sur un sujet
    sélective : une partie des sources selon des critères

  • Bibliographie commentée par REFAD

    • selon le dépôt légal

Norme ISO 690
norme internationale (4e édition)

  • Ordre des informations :
    • 1. Nom et Titre avec indication du support si nécessaire / + date
    2. Édition, lieu, éditeur, titre collection, numéro, volume
    3. Numéro normalisé : ISBN, ISSN, DOI
    4. Autres info dont la localisation
  • Informations nécessaires selon les types de publications
    • pour un livre : nom, titre, support si pas imprimé, édition si pas première, date si mode Nom-Date, date de citation pour ressource en ligne, etc.
  • Variations selon les domaines

Numéro normalisé

  • ISBN - International Standard Book Number (ISSN pour articles)
    • composition : numéro du pays = 978 - groupe linguistique - numéro d'éditeur - numéro de publication - clef de contrôle

Identifiants numériques

  • URI (Uniform Resource Identifier)
    • URL et URN
  • DOI (Digital Object Identifier) par Association of American Publishers
    • alternative à l'URI
    demande auprès de Crossref pour les publications académiques
    avec des frais d'abonnements : 1 $/DOI article ; 2-3$/DOI livre

Alternatives

  • ARK - Archival Resource Key (California Digital Library)
    • moins centralisé mais gratuit - plus souple que DOI
    utilisé par la BNF (Gallica) et plusieurs autres
    exemple : gallica.bnf.fr/ark:/...
  • ASIN - Amazon Standard Identification Number
    • utilisé et produit par Amazon
  • LCCN ou Library of Congress Control Number
    • par la Bibliothèque du Congrès pour catalogage

Créer une bibliographie (export) - En local

  • 1. Clic droit sur votre collection
  • 2. Sélectionner l'option « Créer une bibliographie »
  • 3. Choisir le style (APA 7e) et la langue (Français)
  • 4. Choisir l'option Bibliographie et « Copier sur le clavier »
  • 5. Cliquer sur « Ok »
  • 6. Dans un fichier docx, coller (CTRL+V)

documentation

Éditeur de texte
dans Stylo - citer

  • Dans le corps du texte - ajouter citation
    • en copiant-collant depuis le volet bibliographie
    ou en utilisant la fonctionnalité de citation automatique, en tapant « @ »
    la citation doit apparaître sur ce modèle : [@nom_titre_date]
  • Plusieurs modes de citation
    • [@nom_titre_date, p. 123] en aperçu donnera (Nom date, 123)
    [-@nom_titre_date] en aperçu donnera (date)
    [@nom_titre_date ; @nom_titre_date] en aperçu donnera (Nom date ; Nom date)
    Prénom @nom_titre_date[p. 123] en aperçu donnera Nom (date, 123)
    Nom [-@nom_titre_date ; -@nom_titre_date] en aperçu donnera Nom (date ; date)
